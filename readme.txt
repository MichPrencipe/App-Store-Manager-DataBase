Il progetto è realizzato grazie alla connessione tra JDBC e mySQL.

L'interfaccia grafica è stata realizzata con la libreria JavaFX,i quali moduli sono stati scaricati ed aggiunti come argomento alle configurazioni di run. E' indispensabile,quindi aggiungere tali moduli prima di runnare l'applicazione.

Le credenziali sono gestite dal nostro database tramite una query. Se si è già registrati all'appStore, basta inserire il proprio id,username,password per accedere alle relative operazioni di utente o autore.
